# Donkey Kong (Barrel Rescue)

![Donkey Kong (Barrel Rescue)](./17_donkey_kong.jpg)

文件: [17_donkey_kong.ino](./17_donkey_kong.ino)

## 操作
- 摇杆左右: 移动
- 摇杆上下: 靠近梯子时爬梯
- 蓝键: 跳跃

## 玩法
- 躲避滚桶并到达顶层旗帜。
- 过关后难度提升；生命归零进入 Game Over 与排行榜。

## 可调参数（含代码行）
- 玩家移动/爬梯/跳跃速度 `PLAYER_MOVE_SPEED` / `PLAYER_CLIMB_SPEED` / `PLAYER_JUMP_SPEED`（37-39）
- 重力 `GRAVITY`（40）
- 滚桶基础速度 `BARREL_SPEED_BASE`（43）
- 滚桶上限 `MAX_BARRELS`（28）
- 排行榜大小 `LEADERBOARD_SIZE`（29）
